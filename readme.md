# Express Jobly

A backend job board application built with Node.js, Express, PostgreSQL, and JWT authentication.

## Features

- JWT authentication
- Admin authorization
- Companies CRUD
- Jobs CRUD
- Job applications
- Dynamic filtering
- JSON Schema validation
- PostgreSQL relational database
- Jest and Supertest testing

---

## Technologies Used

- Node.js
- Express.js
- PostgreSQL
- JWT
- bcrypt
- Jest
- Supertest
- JSON Schema
